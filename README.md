## 🚀 About

KlippyDash is a modern desktop dashboard for Klipper-powered 3D printers, designed to provide a cleaner, faster and more immersive experience than traditional browser-based interfaces.

Built with a strong focus on usability, performance and futuristic UI design, KlippyDash transforms printer management into a modern control-center experience.

---

## ✨ Features

- Real-time printer monitoring
- Modern dashboard interface
- Klipper + Moonraker integration
- GCode console
- Macro management
- File browser
- Multi-printer support
- Touch-friendly interface
- Portable Windows application
- Dark / Neon UI themes
- Fast lightweight desktop experience

---

## ⚠️ Beta Disclaimer

KlippyDash is currently in beta stage.

Some features may still be experimental or incomplete, and unexpected bugs or compatibility issues may occur depending on your printer setup or Moonraker configuration.

Use at your own risk.
